# Cedros Login

> **Warning:** This project is experimental and under active development. APIs may change without notice. Not recommended for production use.

Authentication library for React applications with email/password, OAuth, passkeys, and Solana wallet support.

## Packages

- **[@cedros/login-react](./react)** - React component library
- **[@cedros/login-sidecar](./sidecar)** - Privacy Cash sidecar service
- **[cedros-login-server](./server-rust)** - Rust authentication server

See individual package READMEs for documentation.
